### 🧪 실습 1. 포모도로 타이머 만들기 (CLI 버전)

### 🎯 실습 목표

파이썬의 `time` 모듈과 반복문, 함수 정의를 이용해

**CLI(터미널 기반) 포모도로 타이머**를 직접 만들어봅니다.

---

### 📌 요구 사항

- `25분 집중 → 5분 휴식` 사이클을 **4번 반복**
- 마지막에는 **15분 긴 휴식**
- 사용자는 `Enter`를 눌러 시작
- 각 타이머가 끝나면 **문구 출력으로 알림**
- 실제 시간 흐름은 `time.sleep()`으로 구현
- 코드에 간단한 **주석 설명 포함**

---

### 🧠 실습용 프롬프트

```python
# 파이썬으로 CLI 기반의 포모도로 타이머를 만들어줘.
# 25분 집중, 5분 휴식 사이클을 4번 반복하고, 이후엔 15분 긴 휴식이 포함돼.
# 사용자는 Enter를 눌러 타이머를 시작할 수 있게 하고,
# 타이머가 끝나면 알림 메시지를 출력해줘 (소리는 필요 없어).
# 시간은 실제로 기다리는 걸로 (time.sleep 사용) 구현해줘.
# 코드에 간단한 설명도 달아줘.
```

- 이 프롬프트를 Cursor GPT 패널 또는 파일 상단 주석에 작성하면 AI가 자동으로 타이머 코드를 생성해줍니다.
    - Mode: Agent
    - Model: GPT4.1

---

### ✅ 포인트 정리

| 요소 | 설명 |
| --- | --- |
| `time.sleep()` | 실제 시간 지연을 위한 함수 |
| `for i in range()` | 반복 구조 |
| `if i == cycles:` | 마지막 루틴에서 긴 휴식 분기 |
| `input()` | 사용자 입력 대기 (Enter 키) |
| `print()` | 진행 안내 및 마무리 메시지 |
