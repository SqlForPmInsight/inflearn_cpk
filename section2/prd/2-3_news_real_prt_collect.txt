[기능명]
RSS 피드 기반 뉴스 요약 자동화 기능

[배경]
- https://www.mk.co.kr/rss/50300009/ 에서 제공하는 RSS 피드 데이터를 수집
- 기사 본문을 추출한 후 GPT API에 요청하여 요약 결과 생성
- 추후 요약 데이터를 저장하거나 뉴스레터/대시보드 등에 활용 가능성을 염두에 둠

[기능 요구사항]
1. RSS 피드 수집
   - https://www.mk.co.kr/rss/50300009/ 에 주기적으로 접근하여 최신 기사 목록 수집
   - RSS 항목에서 `title`, `link`, `pubDate` 추출

2. GPT 요약 요청
   - OpenAI GPT API에 기사 본문을 전송하여 요약 텍스트 요청
   - 프롬프트는 다음 구조 유지:
     ```
     다음 뉴스 기사를 top 5개 이슈를 간결하게 요약해줘:
     {기사 본문}
     ```
   - 요약 결과와 함께 원 제목(title), 링크(link), 발행일(pubDate)도 함께 저장

3. 출력 형식
   - 결과는 print로 출력하고 txt파일로 저장

4. 실행 방식
   - 초기 버전은 단일 스크립트로 수동 실행
   - 추후 크론탭 또는 워크플로우 자동화(n8n 등)와 연동 고려

[비고]
- GPT 호출 시 토큰 초과 방지를 위한 길이 제한 고려
- 향후 다국어 확장, 요약 스타일 설정 등 기능 고도화 가능성 있음
